# Ionic の概要

Ionic は、Web 技術を使ってモバイルとデスクトップアプリを作るためのフレームワークです。

「Web Components」という Web 標準の技術が採用されており、本書で利用する Angular というアプリケーションフレームワーク
だけではなく、React や Vue.js からも利用することができます。

https://ionicframework.com/docs/components

## 特徴① 使いたくなる iOS / Android 別の美しい UI デザイン

Ionic には、採用している多くのユーザが「UI デザインが洗練されているので Ionic を使う」と応えるような
素晴らしい UI コンポーネントが揃っています。

Ionic の基幹は、再利用可能な UI コンポーネントのライブラリです。

## 特徴② 軽量で高速

Ionic は、「Stencil」という Web Components 開発ライブラリによって作られており、バンドルサイズはとても小さくなります。

また、Ionic の UI コンポーネントは Lazy Loading（遅延読み込み）で読み込まれます。

## 特徴③ モバイルライクな URL ルーティング

モバイル UI を実現するために、Web アプリケーションでは珍しい「非線形ナビゲーション」と
「ナビゲーションスタック」を実現しています。

非線形ナビゲーションは、タブの付いたアプリでよく見かけるタイプのナビゲーションです。

ナビゲーションスタックは、ページ遷移時に前に表示していたページを削除するのではなく、
スタック（一時的に退避）することです。このことによって、戻るときにスムーズにアニメーションを表示することができます。

## 特徴④ モバイルアプリに変換する機能が標準装備

Ionic は、「Capacitor」と「Apache Cordova」という Web アプリをハイブリッドにコンパイルする仕組みをサポートしています。
これらを利用することによって、 Web アプリをさまざまなプラットフォームでリリースできます。
